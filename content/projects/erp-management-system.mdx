A comprehensive Enterprise Resource Planning solution designed to streamline and automate core business operations with a modern, scalable architecture.

## Key Features

- **Role-Based Access Control**: Granular permissions for Admin, Sales, Purchase, and Inventory roles
- **Product Management**: Full CRUD operations for managing product catalog
- **Customer & Supplier Management**: Centralized contact and vendor database
- **Sales & Purchase Orders**: Complete order workflow from creation to fulfillment
- **Inventory Tracking**: Goods Receipt Note (GRN) management and stock tracking
- **Invoice Generation**: Automated invoicing linked to orders
- **Analytics Dashboard**: Visual charts and key business metrics
- **API Documentation**: RESTful APIs with Swagger/OpenAPI documentation

## Tech Stack

- **Frontend**: React.js, Redux Toolkit, Material UI
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **Security**: JWT Authentication, bcrypt password hashing

## Architecture

The application follows a modular architecture with:

- Redux Toolkit for state management
- RESTful API design with proper error handling
- Secure authentication middleware
- Role-based route protection
