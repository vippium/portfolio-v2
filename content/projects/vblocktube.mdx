## Features

- **Custom Blocking Filters**: Create rules to block videos by channel, title, or keywords
- **Flexible Pattern Matching**: Support for regex patterns and wildcard matching
- **Easy Configuration**: User-friendly interface to manage your blocking rules
- **Persistent Storage**: Your preferences are saved locally
- **Active Maintenance**: Continuously updated on GitHub

## Tech Stack

- **JavaScript**: Core script functionality
- **YouTube API**: Integration with YouTube DOM and features
