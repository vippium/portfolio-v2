## Running Locally

Clone the repository:

```bash
git clone https://github.com/vippium/portfolio.git
cd portfolio
```

Install dependencies:

```bash
pnpm install
```

Run the development server:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build for Production

```bash
pnpm build
pnpm start
```
